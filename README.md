# Doseko

> Manage your medication with ease.

Doseko is a mobile health application designed to help users take control of their health by managing medications, tracking side effects, and scheduling medical appointments.

## Key Features

- **Medication Management:** Add, track, and get reminders for medications.
- **Appointment Scheduling:** An integrated calendar to manage medical appointments.
- **Activity Logging:** A simple log for tracking side effects and symptoms.
- **Profile & Health:** Manage personal info and track health metrics (BMI, weight, etc.).
- **Secure Authentication:** Secure login and registration for users.
- **In-App Support:** Includes a Help (FAQ) and About page.

## Core Functionality

- **Onboarding & Auth:** A simple 3-step introduction followed by secure user login and registration.
- **Medicine:** Users can add detailed medication info (dose, type, time) and view their daily medication schedule.
- **Appointments:** A full calendar view allows users to add appointments with notes, dates, and times, and see them listed by day.
- **Log Activity:** Users can select a date and check off side effects (like headache, fever, etc.) to create a simple health log.
- **Profile:** A central hub to "Edit Profile" (name, photo), update "Health Details" (BMI, weight), and access the "Help" and "About" sections.
